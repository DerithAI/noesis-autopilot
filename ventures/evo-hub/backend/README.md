# Personalized Evo Learning Path API

## Overview
This is a FastAPI backend service for handling REST and GraphQL APIs related to personalized evo learning paths.

## Usage
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   docker build -t personalized-evo-learning-path .
   docker run -p 8000:8000 personalized-evo-learning-path
   ```

3. Test the API using `pytest`:
   ```bash
   pytest tests/
   ```
```
