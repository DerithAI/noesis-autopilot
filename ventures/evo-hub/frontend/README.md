# Personalized Evo Learning Path Dashboard

## Overview
This is a React-based admin panel/dashboard for the Personalized Evo Learning Path application. It includes components like Header, Sidebar, Cards, Charts, and Tables.

## Getting Started
1. Clone the repository.
2. Install dependencies with `npm install`.
3. Run the development server with `npm start`.
