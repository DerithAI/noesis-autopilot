# 🤖 EVO-Bot

AI assistant with Ollama brain, WOLF_AI pack integration, LUMENA bridge, and Telegram interface.

## Quick Start

```bash
# Chat via CLI
python bot.py chat "Hello world"

# Interactive mode
python bot.py interactive

# Check status
python bot.py status
```

## Commands

| Command | Description |
|---------|-------------|
| `chat <msg>` | Send message to AI |
| `interactive` | Run interactive chat session |
| `status` | Show bot status |
| `tool <name> <cmd>` | Execute tool hook |
| `set-personality <name> <desc>` | Change personality |

## Tools

### WOLF_AI Bridge

```bash
python bot.py tool wolf "status"
python bot.py tool wolf "howl Hello pack"
python bot.py tool wolf "hunt target"
```

### IMPULSE Bridge (LUMENA CORE)

```bash
python bot.py tool impulse "status"
python bot.py tool impulse "chat Hello LUMENA"
```

## Telegram Bot

### Setup

1. Talk to @BotFather on Telegram
2. Create bot: `/newbot`
3. Run setup:
   ```bash
   python setup_telegram.py
   ```
4. Or manually create `.env`:
   ```
   TELEGRAM_BOT_TOKEN=your_token_here
   TELEGRAM_ALLOWED_USERS=your_telegram_id
   OLLAMA_MODEL=qwen2.5:7b
   ```

### Run

```bash
python setup_telegram.py --run
# or directly:
python telegram_bot.py
```

### Telegram Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/chat <msg>` | Talk with AI |
| `/status` | Bot status |
| `/wolf <cmd>` | WOLF_AI bridge |
| `/impulse <cmd>` | LUMENA bridge |
| `/help` | Show help |

Any plain text message is automatically treated as AI chat.

## Configuration

Copy `.env.example` to `.env` and configure:

- `OLLAMA_HOST` - Ollama server (default: 127.0.0.1:11434)
- `OLLAMA_MODEL` - Model to use (default: qwen2.5:7b)
- `TELEGRAM_BOT_TOKEN` - From @BotFather
- `TELEGRAM_ALLOWED_USERS` - Comma-separated user IDs
